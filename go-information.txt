1. initilize go projext : go mod init github.com/bigzirook/movie-ticket-booking (project name)
2. initilize gin package : go get -u github.com/gin-gonic/gin
3. installing jwt : go get github.com/dgrijalva/jwt-go
4. run go project : go run main.go
5. install all neccesery package : go mod tidy
6. go get -u github.com/go-sql-driver/mysql
